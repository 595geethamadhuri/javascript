# javascript
### function
function without parameter
```javascript
function addition(){
    var a=10;
    var b=30;
    return a+b;
}
addition()
40
```
function with parameter
```javasript
function addition (a,b){
    return(a+b)
}
addition (2,4)
6
```
function substraction(a,b){
    return(a-b)
}
substraction(6,3)
3
```

var substraction=function(a,b){
